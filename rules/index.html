Sudoku Rules

Sudoku is a puzzle game played on a 9×9 grid, divided into 3×3 subgrids called regions.

  • The goal is to fill all empty cells with numbers from 1 to 9.
  • Each row must contain the numbers 1 to 9, with no repeats.
  • Each column must also contain the numbers 1 to 9, with no repeats.
  • Each 3×3 region must also contain the numbers 1 to 9, with no repeats.
  • The puzzle begins with some numbers already filled in (called clues).
  • The challenge is to use logic to fill in the remaining numbers correctly.

Tips for Beginners

  • Start with rows, columns, or regions that already have many numbers filled in.
  • Look for “single candidates” — places where only one number can fit.
  • Practice often! Sudoku gets easier the more you play.
